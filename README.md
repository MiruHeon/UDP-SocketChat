# UDP-SocketChat

Python의 socket 라이브러리를 활용한 UDP 기반 간단한 채팅 시스템입니다.
UDP 프로토콜의 특징을 직접 경험하고, 비연결성 통신 방식에 대한 이해를 높일 수 있습니다.

## 주요 기능

실시간 메시지 송수신: 클라이언트와 클라이언트간에 간단한 텍스트 메시지를 전송 가능합니다.<br>
다중 사용자 지원: 여러 클라이언트의 동시 메시지 전송이 가능합니다. (데몬스레드 사용)<br>

## 기술적 특징

UDP 프로토콜 사용: 연결 설정 없이 빠른 데이터 전송 가능, 전송 신뢰성은 낮음
비동기 처리 가능: threading 모듈을 활용해 메시지 송수신을 동시에 처리
경량화: 별도의 서버 설치 없이 간단히 로컬 테스트 가능
